URL Shortener API

An API that allows the user to turn a regular URL into a shortened one for ease of replication and use. The API will prepend a https:// if your input doesnt't contain it, but otherwise it's up to you to ensure the URL is valid.

After adding a URL to the Database, the API will return a JSON object of the form {"_id" : "a1e2f0", "originalUrl" : "https://www.wherever.com"}

API end-points:

Add a URL: url/new/www.newurl.com

Link to the URL: url/a1e2f0
